Dedicats [Server]

  • By Codestake Team
  • 02, 2020

길고양이 돌봄 커뮤니티, 모바일 애플리케이션입니다.

Stacks

  • Express
  • TypeScript
  • TypeORM
  • JWT
  • AWS_EC2
  • mobX
  • MariaDB

Contribution

  • JWT이용, access token, refresh toeken의 개념으로 자동로그인, 로그인 유지 구현
  • MVC패턴과 비즈니스 로직을 담당하는 Service 레이어로 관심사를 명확히 분리하여 유지 보수 용이성과 코드 가독성 향상
  • 일괄적인 에러와 예외 핸들링을 하도록 리펙토링하여 디버깅 시간 감소 및 클라이언트와의 효율적인 의사소통
  • 유저, 게시글, 댓글부분 CRUD 구현. 고양이 정보 부분에서 Spatial Query(공간쿼리문)활용
  • 회원가입, 로그인, 로그인 유지 및 유저 인증 절차를 위한 클라이언트 MobX 사용 및 react-nativee 경험
  • 데이터 베이스 주도적 작성, API문서 플랜 및 작성

Server - File Tree

MVC 패턴을 베이스로 (router, Controller, Service, Model) 구성하였습니다.
library는 모듈화 된 파일들이, types에는 enum type들이 존재합니다.

API문서

시연

회원가입

유효한 이메일 주소를 입력하면, 인증 이메일이 전송됩니다. 코드가 일치 할 때 회원가입이 승인됩니다.

로그인

최초 로그인 후 로그인 유지가 됩니다.
성공적으로 로그인 후 서비스 메인화면으로 안내됩니다.

고양이 등록

사진을 업로드 하고, 고양이 별명과 종, 설명, 중성화 여부를 선택하면 고양이가 추가됩니다.
정상 등록 후엔 등록 위치에서 고양이를 찾을 수 있습니다.

고양이 BIO

오늘의 건강상태를 업데이트 하고, 해시 태그를 입력 할 수 있습니다.
중성화여부에 투표 할 수 있고, 고양이가 보이지 않는다면 실종 신고도 할 수 있습니다.

고양이 게시물

고양이 바이오에서, 고양이 게시물과 게시물에 댓글을 달아 이용자들과 의사소통 할 수있습니다.
댓글은 socket.io로 구현되어 실시간 소통이 가능합니다.

마이페이지, 문의하기, 비밀번호 변경

마이페이지에서 유저의 프로필을 변경하고, 팔로잉 중인 고양이를 확인 할 수 있습니다.
비밀번호 수정과 문의하기, 로그아웃이 가능합니다.

비밀번호 찾기

비밀번호를 잃어버렸을 경우, 임시비밀번호를 이메일로 재 발급 받을 수 있습니다.
임시비밀번호로 로그인시, 비밀번호를 변경페이지로 안내됩니다.
