Framer Recipe

  • By Team.어머니호구마
  • 07, 2019

농산물 직거래 및 농부의 블로깅 사이트를 만든 프로젝트 입니다.

Stacks

  • Java
  • JSP
  • JQuery, AJAX
  • JDBC
  • Oracle DB
  • Apache Tomcat

Contribution

  • 근마이페이지 전체 페이지 및 기능 구현 ( 농부 팔로우 , 게시글 좋아요 기능 , 내 정보 관리 및 수정 , 적립금 조회 , 주문 내역 조회 기능 )
  • 프로젝트 전체 구조 설계 및 DB 스키마 설계
  • 회원은 DB 를 비회원은 쿠키를 이용하는 방식으로 장바구니 구현
  • 카카오 로그인 API, 카카오 지도 API 를 활용 , 적극 커스텀 하여 농장 위치 표기 , 사용자 경험을 끌어올리고자 함
  • JQuery 와 CSS3 스펙을 이용하여 이마트 장바구니 , 쿠팡 상품 페이지 클로닝
  • 전체 UI 기획, 구상, 디자인
